EventRepositoryImp

class EventRepositoryImp(db: FirebaseFirestore, userRepository: UserRepository, eventDao: EventDao, cityDao: CityDao, participantResponseDao: ParticipantResponseDao, restaurantDao: RestaurantDao, placesRepository: PlacesRepository, auth: FirebaseAuth) : EventRepository

Implementation of EventRepository responsible for creating events and storing them in Cloud Firestore.

This class also coordinates with UserRepository to update the host's created event list after a successful event creation.
