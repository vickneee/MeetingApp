ParticipantViewModel

class ParticipantViewModel(eventRepository: EventRepository, savedStateHandle: SavedStateHandle) : ViewModel

ViewModel for the participant input flow.

Responsibilities:

  • Loads event data using eventCode and eventKey

  • Tracks participant input (name, availability, locations, place types, food categories)

  • Observes submissions and determines whether the user has already submitted

  • Exposes UI state for availability selection, place preferences, and submission status

  • Provides helper functions to toggle selections and submit participant input

The ViewModel is scoped to the participant-input navigation graph, ensuring that participant input persists across multiple screens in the flow.
